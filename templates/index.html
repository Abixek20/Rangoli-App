📖 About Rangoli AI

A GenAI-powered prototype exploring computational folk art through data structures and algorithms.

🧮

Geometric Engine

Patterns are generated using matrix-based geometric transformations — rotation matrices, reflective symmetry operations, and parametric curve equations (Archimedean spirals, rose curves). The engine produces n-fold radial symmetry with configurable layers.

NumPy Pillow Matrix Math
🕸️

Graph Theory Model

Every Rangoli pattern is represented as a graph — dot grid points become nodes, and Delaunay triangulation creates edges preserving geometric adjacency. NetworkX is used to compute density, degree distribution, and estimated symmetry order.

NetworkX Delaunay SciPy
🔬

Computer Vision

The analyzer uses OpenCV for edge detection (Canny), Hough line/circle transforms, and contour classification. K-Means clustering (sklearn) extracts the dominant color palette from any uploaded Rangoli image.

OpenCV Canny Edge K-Means
🌺

Pattern Styles

Five drawing algorithms — Mandala (concentric petals), Kolam (Bézier curves on a dot grid), Floral (lotus motifs), Geometric (star polygons), and Peacock (feather-eye motifs with Archimedean spirals).

5 Styles 5 Color Schemes Customizable
⚙️

Tech Stack

Built with Flask for the backend REST API, PIL/Pillow for image rendering, and vanilla JavaScript on the frontend. All pattern generation is deterministic — driven purely by algorithmic geometry with no neural networks required.

Flask Python 3 REST API Vanilla JS
🎓

Academic Context

This project demonstrates how Data Structures & Algorithms (graph representations, matrix operations, spatial data structures) and Machine Learning concepts (clustering, feature extraction) can be applied to preserve and recreate traditional art forms.

DSA ML Folk Art

🔌 API Reference

Three REST endpoints power the application

POST
/api/generate

Accepts style, symmetry, layers, colorScheme, complexity. Returns base64 PNG + graph statistics.

POST
/api/analyze

Accepts multipart image upload. Returns symmetry scores, color palette, shape features, and edge map.

GET
/api/styles

Returns all available styles, color schemes, symmetry options, and configuration metadata.
